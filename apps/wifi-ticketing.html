Features

Everything to Run
Your Hotspot

🎟️

Generate Vouchers Instantly

Create WiFi access vouchers in seconds. Set custom time limits — 1 hour, 3 hours, 24 hours, 7 days, or monthly packages. Each voucher gets a unique code.

📲

Print or Share via WhatsApp

Print physical voucher tickets for your customers, or send the code directly via WhatsApp in seconds. Works with standard receipt printers.

👁️

Monitor Active Sessions

See all currently active sessions in real time — who's connected, how much time is remaining, and which voucher they're using. Kick sessions instantly if needed.

💼

Multiple Voucher Packages

Set up different pricing tiers — hourly, daily, weekly. Each package has its own price, time limit, and data limit. Customers choose what suits them.

📊

Revenue & Usage Reports

Track how many vouchers were sold, which packages are most popular, and your daily revenue — all from one dashboard.

🔐

Secure Access Control

Only customers with valid vouchers can connect. Expired or used vouchers are automatically blocked, keeping your network secure and monetised.
